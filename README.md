## 💡 Troubleshooting common issues

### The application does not start
Ensure you have the latest version of the .NET runtime installed from the official Microsoft support page. Restart your computer if the installer hangs.

### The AI shows connection errors
This means the selected pathway is currently busy or offline. Switch to a different proxy address in the settings menu. Many pathways cycle through states of availability throughout the day. 

### Windows blocks the file
Windows Defender sometimes prevents unknown software from running. This is a common precaution for new tools. If you trust the source, click Run anyway in the alert box. 

### The connection is silent
Check your internet firewall settings. Sometimes firewalls detect the proxy activity and block the outgoing signal. Add this application to your firewall whitelist to grant it permission to access the web.
